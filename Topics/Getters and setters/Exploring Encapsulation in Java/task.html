Exploring Encapsulation in Java

In the context of Java, consider the following code snippet:


 public class Person {
     private String name;
     private int age;

     public String getName() {
         return this.name;
     }

     public void setName(String name) {
         this.name = name;
     }

     public int getAge() {
         return this.age;
     }

     public void setAge(int age) {
         if (age > 0) {
             this.age = age;
         }
     }
}
