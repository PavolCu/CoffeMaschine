The Int class

Here's a simple wrapper class for values of the int type.

public class Int {

    int val = 1;

    public Int() {
        val = 2;
    }

    public Int(int val) {
        this();
    }
  
    public Int(int val1, int val2) {
        this(val1 + val2);
    }
}

What is the value of the field val in the created object?
